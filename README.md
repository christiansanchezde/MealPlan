A cross-platform mobile and desktop application designed to streamline the meal planning and grocery shopping process. Built as a learning project to explore .NET MAUI development.

### 🌟 Motivation

This project was born out of a desire to replace a manual, inefficient process. Previously, managing a cooking rotation, building a weekly plan, and organizing groceries was handled manually using Microsoft To Do. This application automates these tasks, organizing the grocery list efficiently and saving time during supermarket visits.

### 🛠 Technology Stack

  * **Framework:** .NET MAUI
  * **Database:** SQLite (local storage)
  * **Architecture Pattern:** MVVM (Model-View-ViewModel)
  * **Toolkit:** MVVM Community Toolkit

### 📋 Features

  * **Recipe Management:** Input, store, and manage your favorite cooking recipes in a central database.
  * **Weekly Planning:** Select meals for your weekly plan easily.
  * **Automated Grocery List:** The application automatically generates a comprehensive grocery list based on the ingredients required for your selected weekly plan.
  * **Smart Organization:** Your generated grocery list is automatically organized by supermarket sections, making shopping faster.
  * **Digital Checklist:** Track your progress while shopping with a digital checklist to check off items as they land in your cart.

### 📂 Project Structure

The solution contains two main components:

1.  **`MealPlanner.App` (MAUI):** The main cross-platform mobile application, containing the user interface, business logic, and database interactions, built with the MVVM Community Toolkit.

### 🚀 Getting Started

1.  Clone this repository.
2.  Open the solution file (`MealPlanner.sln`) in Visual Studio (used the community version).
3.  Ensure you have the .NET MAUI workload installed.
4.  Restore NuGet packages.
5.  Set `MealPlanner.App` as the startup project.
6.  Run the application on an emulator or a physically connected device (Android, iOS, Windows).
